# ALL_CIPHERS

With this program, you can Encrypt and Decrypt any Text that you will give it using 10 different Ciphers

1- Affine Cipher
2- Caesar Cipher
3- Atbash Cipher
4- Vignere Cipher
5- Baconian Cipher
6- Simple Substitution Cipher
7- Polybius Square Cipher
8- Morse Cipher
9- XOR Cipher
10- Rail-fence Cipher
